# Number-guessing-game
A CLI based number guessing game in C where the user guesses a randomly generated number with hints and attempt tracking.

# Number Guessing Game in C

A simple command-line Number Guessing Game written in C.

## Features
- Random number between 1 and 100
- Hints (Too high / Too low)
- Counts number of attempts

## Technologies Used
- C Programming
- Standard Library (stdlib, time)

## How to Run
1. Compile:
   gcc main.c -o game
2. Run:
   ./game

## Example
Enter your guess: 50  
Too low!  
Enter your guess: 75  
Too high!  
Enter your guess: 63  
Correct!
